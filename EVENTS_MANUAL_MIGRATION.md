## Step-by-Step Instructions

### Via Admin Panel UI

1. **Log in to Admin Panel**
   - Go to: `https://sikhaidglobal.jivomedia.com/admin/`
   - Email: `admin@sikhaidglobal.com`
   - Password: (Your Firebase password)

2. **Navigate to Events**
   - Click "Fundraising Events" in left sidebar

3. **Add Each Event**
   - Click "+ Add Event" button
   - Copy data from the table above
   - For **Event Image**: Copy the image URL provided
   - Leave **Registration Link EMPTY** for all old events
   - Click "Save Event"

4. **Verify**
   - Go to main website: `https://sikhaidglobal.jivomedia.com/`
   - Scroll to "News & Events" section
   - All 5 events should appear with images and details

### Important Notes

✅ **DO:**
- Copy exact event names
- Use correct dates (format: YYYY-MM-DD in date picker)
- Include full descriptions
- Use the image URLs provided (already hosted on Unsplash)
- Leave Registration Link EMPTY for all old events

❌ **DON'T:**
- Add registration links for old/completed events
- Change event names or descriptions
- Leave description empty
- Skip any events

## Troubleshooting

**Events not showing on website?**
- Hard refresh browser (Cmd+Shift+R)
- Wait 30 seconds for cache to clear
- Check Firebase Firestore → events collection

**Images not loading?**
- The URLs should work as-is (Unsplash CDN)
- If image fails, leave it blank (default placeholder will show)

**Admin panel not saving?**
- Check browser console for errors
- Verify all required fields are filled (*)
- Ensure Firebase Storage rules allow uploads
